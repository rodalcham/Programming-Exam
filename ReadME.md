# Hi Antonio!

I've chosen to implement only the two first matrix structures, as I found quad-tree to be really complex.
Regardless, I think I did a good job with the rest of the requirements.

You can run this command to execute the program:

```bash
make run
```
If you have any issues, try creating the ```obj``` directory manually, or changing privileges to allow the creation of directories.

If you would like to take a closer look into the output, you can copy this command:

```bash
make log
```

It will direct all the output into an out.txt file.
